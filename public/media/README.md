# Media folder

Store your portfolio media files here.

## Files used by the app

- `cv.pdf` - shown in the CV viewer
- `application-letter.pdf` - shown in the application letter viewer
- `profile-showcase.svg` - hero image shown near the top of the page

## Replace with your real files

You can replace the placeholder PDFs and image with your own files while keeping the same filenames.

## Optional additions

You can also place other files here later, for example:

- certificate PDFs
- profile photo
- project screenshots
- achievement images
